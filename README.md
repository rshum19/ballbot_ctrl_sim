# Ballbot Matlab Control Simulation
Matlab simulation of the CMU Ballbot

## Install 
1. Move ot the direction `ballbot_ctrl_sim`
2. Run the script `install_ballbot_sim.m`. 

## Dependencies
| Name | Description|
